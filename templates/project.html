001 : Air Quality Index Prediction Using ML


Team Members:

  • Satishkumar
  • Rishit
  • Sudarshan
  • Manish

Project Summary:

It is a Flask web application that utilizes machine learning models to predict air quality based on input parameters such as PM2.5, PM10, O3, NO2, CO, and SO2 values.

Objective:

Developed a web application for predicting the Air Quality Index (AQI) based on a machine learning model. The project aimed to compare and analyze the performance of three different regression models: Decision Tree, Random Forest, and Linear Regression. The model with the highest R-squared score, which happened to be the Random Forest model, was chosen for deployment.

Machine Learning Models:

Decision Tree Model:
  1. Max Depth: 10
  2. Min Samples Split: 10
Random Forest Model:
  1. Depth: 20
  2. Min Samples Split: 10
  3. Number of Estimators: 200
Linear Regression Model

Web Application:

  • Built using Flask, a Python web framework.
  • The application takes input values for PM2.5, PM10, O3, NO2, CO, and SO2 from users.
  • Upon submission, the web app uses the trained Random Forest model to predict the AQI.
  • The predicted AQI is then categorized into different air quality levels (e.g., Good, Satisfactory, Moderately Polluted) with corresponding health impact conclusions.
  • Results are displayed to the user on a separate results page.

Model Evaluation:

  • R-squared scores were calculated for each model to assess their predictive performance.
  • A bar graph was created to visually compare the R-squared scores of the Decision Tree, Random Forest, and Linear Regression models.

Conclusion:

Thus my team, successfully implemented machine learning models for AQI prediction. The Random Forest model, with the highest R-squared score, was selected for deployment in the web application. This decision was based on the model's superior predictive accuracy compared to the other considered models. The interactive web interface allows users to obtain real-time AQI predictions and health impact assessments. This project demonstrates collaborative teamwork, machine learning expertise, and effective communication of results through a user-friendly web application.
